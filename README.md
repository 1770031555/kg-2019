# kg-2019-final
2019年百度的三元组抽取比赛（ http://lic2019.ccf.org.cn/kg ），“科学空间”队源码，最终测试集第8名，F1为0.8807。

基于“CNN + Attenton + 自行设计的标注结构”的信息抽取模型。

标注结构是自己设计的，我看了很多关系抽取的论文，没有发现类似的做法。所以，如果你基于此模型做出后的修改，最终获奖了或者发表paper什么的，烦请注明一下（其实也不是太奢望）
